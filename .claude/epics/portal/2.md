# Task: Next.js スキャフォールド + 静的エクスポート設定 + 共通レイアウト

## Description

Next.js 15 (App Router / TypeScript / Tailwind CSS v4) プロジェクトを手動スキャフォールドし、静的エクスポート設定と全ページ共通のレイアウト（ヘッダーナビ + フッター）とデザイントークンを実装する。

## Acceptance Criteria

- [ ] `package.json`（next / react / react-dom / tailwindcss / typescript）が作成され `npm install` が成功する
- [ ] `next.config.ts` に `output: 'export'`, `basePath: '/portal'`, `images.unoptimized: true` が設定されている
- [ ] `app/layout.tsx` に ai-foo ロゴ + 4 ページ（ダッシュボード/部署/名簿/FAQ）へのナビゲーションとフッターがある

- [ ] 仮の `app/page.tsx`（プレースホルダ）で `npm run build` が成功する
